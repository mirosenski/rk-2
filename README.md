# RevierKompass Step2 Prototype

This repository contains a minimal prototype for the second step of the RevierKompass wizard. It is built with **React 19**, **Vite** and **TypeScript**.

## Development

```bash
npm install
npm run dev
```

Open <http://localhost:3000> to view the demo.

The implementation is focused on demonstrating state management with Valtio and a Leaflet-based map. The API route `src/api/calculate-routes.ts` outlines the expected request/response shape for OSRM based routing.
